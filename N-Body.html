Gravitational N-Body Problem

In this report a simulation of free moving particles is carried out using the N-body problem formulation and the symplectic Euler method. The problem as well as the implementation and serial optimization in C code is described. The N-body problem is a set of equations governing the movement and interactions between free moving bodies in an empty space. These equations are infeasible to solve by hand as the complexity grows exponentially with the amount of interacting bodies. Using computer simulations though this is a much more feasible task.
